It uses a small number of well understood hardware components and no software.

It relies on

  1. disconnecting power to the propeller motors controlled by a micro switch (initiator) and a relay.
  2. the propeller rotation speed being governed. This is required because excess speed will prevent the guard reacting in time. This is achieved by a simple pulse counter from a hall sensor and magnet (on the propeller shaft) which removes power to the propeller motors if the count rate is exceeded.

100% test coverage has been achieved.

The design is justified in the Design Justification Report (DJR)
